# 개발 목적
주식 초보자가 투자를 시작할 수 있도록 돕는 사이트

# 사용자
주식 투자 경험이 거의 없는 초보자를 핵심 타깃으로 함.  
가계부를 관리하며 안정적인 재테크를 시작하고 싶어 하는 사용자.  
전문 용어보다는 쉽고 직관적인 표현을 선호하며,  
시각적인 정보를 통해 이해하는 데 도움을 받는 사용자.  

# 받아야할 정보(요구사항)
아이디, 비밀번호
관심 있는 투자 분야 선택 (배당형 / 성장형 / 소비형)

# 사용자에게 전달할 정보
뉴스
주식 그래프
대표 종목 추천

# 로드맵
각각 서비스 UI 구현
서비스를 위한 통신 모듈 개발
유저 개인화를 위한 서버 개발
최종적으로는 UI 통합을 통해 웹페이지 서비스 완성

#   사용 환경 관련 요구사항
업데이트 시점: 실시간
사용 시점: 퇴근 후 저녁 시간, 주말
사용 환경: 모바일&데스크탑,  간단&직관적 UI

로그인 → 투자 성향 선택 → 성향에 따라 추천 주식 출력 → 최신 뉴스 / 주식 정보(그래프) 출력 
*파이썬으로 주가데이터 모듈 및 뉴스 크롤링 모듈 생성*
