## 📌 기능 요약
주어진 디렉토리에서 특정 확장자(`*.json`)를 가진 파일을 찾아 파싱하고, 그 내용을 딕셔너리 리스트로 로드하여 반환합니다. 설정 정보를 관리하는 데 사용됩니다.

## ⚙️ 주요 함수

### 1. _load_single_json_file(filepath: 문자열) (내부 함수)
* **기능**: 지정된 경로의 단일 JSON 파일을 로드하고 파싱합니다.
* **로직**:
    1.  파일을 열고 JSON을 로드합니다.
    2.  파일 없음, JSON 형식 오류 등의 예외를 처리하고 오류 메시지를 출력합니다.
    3.  성공 시 딕셔너리를 반환하고, 실패 시 `None`을 반환합니다.

### 2. load_user_profiles(directory_path: 문자열, extension: 문자열)
* **기능**: 디렉토리 내의 모든 설정 파일을 로드하는 주요 공개 함수입니다.
* **로직**:
    1.  디렉토리 존재 여부를 확인합니다.
    2.  디렉토리 내의 모든 파일을 순회하며, 확장자가 일치하는 파일만 필터링합니다.
    3.  필터링된 파일 경로를 `_load_single_json_file`에 전달하여 데이터를 로드합니다.
    4.  성공적으로 로드된 모든 데이터를 리스트에 모아 반환합니다.
